FINALIZE THE PROJECT REPORT ON WINDOWS

This optional procedure updates the Table of Contents and fields in the final Word report, then exports a PDF.

1. Open docs/Ryan_Final_Project_Report.docx in Microsoft Word.
2. Press Ctrl+A to select the document.
3. Press F9 to update all fields.
4. Right-click the Table of Contents and choose Update Field, then Update entire table.
5. Save the document.
6. Use File > Export > Create PDF/XPS to export the PDF.

The previous PowerShell automation file was removed from the submission repository because email providers such as Gmail block ZIP archives containing .ps1 files. The project application does not depend on that script.
